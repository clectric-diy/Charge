# Potentiometer KiCad Files

KiCad source files for the potentiometer breakout board.

## Key Files

- `Potentiometer.kicad_sch`: schematic (title block: "Charge Potentiometer").
- `Potentiometer.kicad_pcb`: PCB layout.
- `Potentiometer.kicad_pro`: project configuration.
- `Potentiometer.kicad_prl`: local project state.
