# Encoder Breakout

General-purpose breakout directory for a standalone rotary encoder module.

## Purpose

Create a reusable encoder input building block for control-surface and panel UI designs.

This module is aimed at both first-time Synth DIY makers who want a simple way to add knob-based interaction, and experienced designers who want a clean, repeatable encoder front end.

If you want an encoder module with onboard LEDs, see `../Encoder_RGB/`.

## Status

No KiCad design files are currently tracked in this folder yet. Add schematic/PCB assets here as the module is developed.
