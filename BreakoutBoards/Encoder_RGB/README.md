# Encoder + LEDs Breakout (Bourns PEL12)

Breakout board combining a Bourns PEL12 rotary encoder interface with integrated LED support.

## Purpose

Provide a compact human-interface module built around a Bourns PEL12 encoder with LEDs for immediate visual feedback.

The intent is to be beginner-friendly for people learning panel controls, while still being practical for experienced builders who want a known-good encoder + LED submodule.

If you only need encoder input (without LEDs), see `../Encoder/`.
