# WS2811 Breakout

Breakout board for a WS2811-based single RGB LED driver stage (NeoPixel-style control of a discrete RGB LED).

## Purpose

Enable serially controlled RGB lighting with a discrete LED and WS2811 driver as a reusable building block.
