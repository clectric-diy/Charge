# WS2811 KiCad Files

KiCad source files for the WS2811 breakout board.

## Key Files

- `WS2811.kicad_sch`: schematic (title block: "Charge WS2811").
- `WS2811.kicad_pcb`: PCB layout.
- `WS2811.kicad_pro`: project configuration.
- `WS2811.kicad_prl`: local project state.
