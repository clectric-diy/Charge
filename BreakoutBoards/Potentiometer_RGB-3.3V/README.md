# Potentiometer + RGB (3.3V) Breakout

Combined control module that pairs a potentiometer interface with an integrated 3.3V RGB LED section.

## Purpose

Combine analog control and visual state indication on one breakout to reduce wiring and panel footprint in UI-heavy builds.

- [KiCad](KiCad/): project container for the combined potentiometer/RGB breakout.
